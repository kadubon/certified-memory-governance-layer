# CMGL v1.1.2 Release Notes

CMGL v1.1.2 is a stable hardening release. It preserves the documented v1 API and focuses on evidence consistency, formal invariant documentation, adapter contract reliability, and release hygiene.

## Highlights

- Formal invariant and proof-obligation documentation.
- Dev-only property-based tests for canonicalization, ledger verification, strict policy, authority non-bypass, retrieval explainability, and conformance fixtures.
- Stable API symbol metadata and import smoke tests.
- Backend semantics and current-view documentation for add-only, update-capable, delete-capable, tombstone-only, temporal graph, and tool/store-shaped backends.
- Adapter contract documentation for missing dependency behavior, fake-client contract tests, optional live smoke, and external SDK version drift.
- v1.1.2 release checklist retained as historical/internal maintenance documentation.

## Compatibility

- No breaking API changes.
- No runtime dependency increase.
- Hypothesis is a dev-only dependency.
- Core remains local-first, deterministic, offline-testable, no-network, no-secret, and no-LLM.

## Claim Boundary

CMGL proves procedural admissibility under declared evidence, receipts, policies, and ledger verification. It does not prove factual truth, external backend correctness, prompt-injection immunity, distributed consensus, legal/compliance certification, or future continued validity.

## Completed Release Gates

- Local quality checks passed: lockfile, dependency sync, formatting, lint, type check, tests, coverage, canonical validation, schema export, adapter doctor, dry-run adapter live smoke, build, publishability, pip-audit, and clean wheel smoke.
- Release publishing uses PyPI Trusted Publishing / OIDC.
- No long-lived PyPI token path is used.
- Core remains no-network, no-secret, and no-LLM.

## Known Maintenance Items

- Keep PyPI Trusted Publisher settings current.
- Review Dependabot PRs separately.
- Verify repository metadata such as topics, description, and homepage after release.
- Run optional protected live adapter smoke for environments that rely on Mem0 or Graphiti live services.
