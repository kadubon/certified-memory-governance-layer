# CMGL v1.1.1 Release Notes

CMGL v1.1.1 is a documentation and release-maintenance patch. It does not change
the core governance API or runtime behavior.

## Highlights

- PyPI long description source no longer contains pre-publication wording.
- README now states directly that CMGL v1.1.1 is available on PyPI.
- GitHub source installation remains documented as a development path.
- v1.1.0 release notes are retained as historical notes with completed release gates
  separated from known maintenance items.
- GitHub Actions dependencies from the open Dependabot PRs were incorporated into
  the workflow files.

## Installation

```bash
uv add cmgl
```

For development from GitHub source:

```bash
uv add "cmgl @ git+https://github.com/kadubon/certified-memory-governance-layer.git"
```

## Compatibility

- Python: 3.10 through 3.13.
- License: Apache-2.0.
- Package name: `cmgl`.
- Public API: unchanged from the documented v1 stable API.

## Completed Release Gates

- `cmgl` is available on PyPI through Trusted Publishing.
- GitHub source installation remains available for development.
- Release checklist documents are retained as historical/internal maintenance docs.

## Known Maintenance Items

- Keep GitHub repository settings, branch protection, secret scanning, and live adapter
  environment settings under periodic review.
